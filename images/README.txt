IMAGE FOLDER STRUCTURE
======================

images/
  hero/
    hero.jpg         → Main hero background image (recommended: 1200x900px)

  products/
    1.jpg            → Diamond Solitaire Ring
    2.jpg            → Gold Chain Necklace
    3.jpg            → Pearl Drop Earrings
    4.jpg            → Rose Gold Bracelet
    5.jpg            → Sapphire Halo Ring
    6.jpg            → Diamond Tennis Bracelet
    7.jpg            → Emerald Pendant
    8.jpg            → Gold Hoop Earrings

  categories/
    rings.jpg        → Rings category cover
    necklaces.jpg    → Necklaces category cover
    earrings.jpg     → Earrings category cover
    bracelets.jpg    → Bracelets category cover

HOW TO USE IMAGES:
------------------
1. Place your .jpg / .png photos in the correct folders above.
2. In index.html, replace the <div class="img-placeholder"> blocks
   with: <img src="images/products/1.jpg" alt="Product Name" />
3. For hero: replace hero-img-placeholder div with:
   <img src="images/hero/hero.jpg" alt="Hero" />
